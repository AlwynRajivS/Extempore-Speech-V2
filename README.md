# 🎤 Extempore — Speech Assessment Portal

> 🚀 A modern, AI-powered platform to evaluate, analyze, and enhance student speaking skills through structured extempore sessions.

---

## 🌟 Overview

**Extempore** is a complete **Speech Assessment System** designed for educational institutions.
It enables **Admins, Faculty, Students, and Class Sessions** to interact seamlessly with real-time analytics and AI-driven insights.

---

## ✨ Features

### 👨‍🎓 Student Portal

* 🎯 Personalized performance tracking
* ⭐ Category-wise rating (Content, Delivery, Confidence, etc.)
* 📊 AI-powered feedback & improvement suggestions

### 👩‍🏫 Faculty Portal

* 📝 Evaluate student speeches easily
* 🎡 Random topic generator
* 📋 Add remarks and ratings

### 🛡️ Admin Panel

* 📊 Institution-wide analytics
* 📈 Section-wise performance tracking
* 🤖 AI-based insights (Groq API integration)
* 👥 Manage students, faculty, and sessions

### 🎡 Class Session Mode

* 🎤 Real-time extempore session handling
* 🔄 Random student/topic selection
* ⚡ Fast evaluation workflow

## 🧠 AI Capabilities

* 📊 Performance analysis using AI

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/extempore-portal.git
   ```

2. Open:

   ```bash
   index.html
   ```

3. Configure:

   * Replace `GAS_URL` with your Google Apps Script Web App URL
   * Add your **Groq API Key** in Admin Panel
